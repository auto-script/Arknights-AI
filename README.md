# Arknights-AI
明日方舟-自动管理




## Condition
1.jdk 8+
2.Nox 模拟器

##Build

git clone xxxxx

clean buildDependents -x test


## Usage
1.将resources与Arknights-AI-x.x.x.jar放置到统一目录
2.将lib中的opencv_java401.dll拷贝到系统盘的\Windows\System32下
3.java -jar Arknights-AI-x.x.x.jar

## Other
可将running.bat添加到windows的计划任务里面,定时启动
